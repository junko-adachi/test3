GitHub Desktop のダウンロードとインストール(Windows)
https://desktop.github.com/
にてgithubのをダウンロード

githubのサイトにて新しいリポジトリを作成する　repositories　でnewをクリック
「Start a project」ボタンをクリック
要な情報を入力したら「Create repository」ボタンをクリックしてリポジトリを作成しよう。


clone or download　ボタンをおしてパソコンのほうにフォルダをコピーしてもらい作成

https://ferret-plus.com/8498
ローカルから新しくリポジトリを作成し、フォルダにデータをいれる
※ドキュメントのgithubのフォルダ


summaryに記入するとcommit MASTERのボタンが押せるようになるのでクリックするとコミットされる

historyをみるとコミットされたことがわかる

リモートリポジトリとの同期
さて、コミットするだけだとローカルリポジトリにファイルが登録されただけで、リモートリポジトリにはファイルが登録されていない。

push　originのボタン（上部黒）を押す
※黒がpushになっていなければ上にあるrepositoryのボタンをクリックするとpushとpullがある

デスクトップのほうでクローンリポジトリをチェック
すると表示されるのでプッシュ
